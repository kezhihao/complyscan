# Changelog

All notable changes to ComplyScan will be documented in this file.

## [0.1.0] - 2026-02-24

### Added
- Initial MVP release
- License scanning for npm, yarn, pnpm lockfiles
- SPDX license database with 20+ licenses
- License compatibility checker
- SARIF format output for GitHub integration
- GitHub OAuth flow
- GitHub webhook handler
- D1 database schema
- GitHub Actions workflow for users
- CI/CD pipeline for project
- Docker development environment

### Planned
- Secret scanning
- Vulnerability scanning (OSV integration)
- Dashboard UI
- PDF report export
- Stripe payment integration
- Multi-repo support
